# SuperMock - Фронтенд

Фронтенд-часть приложения SuperMock для тренировочных собеседований, разработанная с использованием React и Vite.

## Структура проекта

```
react-frontend/
├── public/              # Статические файлы
├── src/                 # Исходный код
│   ├── assets/          # Изображения и другие ресурсы
│   ├── components/      # React компоненты
│   ├── contexts/        # React контексты
│   ├── hooks/           # Пользовательские хуки
│   ├── App.jsx          # Главный компонент приложения
│   ├── App.css          # Стили для главного компонента
│   ├── index.css        # Глобальные стили
│   └── main.jsx         # Точка входа приложения
├── index.html           # HTML шаблон
├── package.json         # Зависимости и скрипты
└── vite.config.js       # Конфигурация Vite
```

### Основные компоненты

#### `App.jsx`

Корневой компонент приложения, который управляет аутентификацией и отображает соответствующие компоненты в зависимости от состояния аутентификации.

**Состояния:**

- `token` - JWT токен пользователя
- `isAuthenticated` - флаг аутентификации
- `activeTab` - активная вкладка ('login' или 'register')
- `apiData` - данные с сервера

**Функциональность:**

- Отображает форму входа или регистрации для неаутентифицированных пользователей
- Отображает профиль пользователя и менеджер сессий для аутентифицированных пользователей

#### `UserProfile.jsx`

Компонент для отображения информации о пользователе и кнопки выхода.

#### `SessionManager.jsx`

Компонент для управления сессиями собеседований.

**Состояния:**

- `activeView` - активный вид ('list', 'create', 'join', 'feedback', 'results')
- `selectedSession` - выбранная сессия
- `refreshTrigger` - триггер для обновления списка сессий

**Функциональность:**

- Отображает список сессий
- Позволяет создавать новые сессии
- Позволяет присоединяться к существующим сессиям
- Позволяет оставлять обратную связь
- Позволяет просматривать результаты обратной связи

### Компоненты аутентификации

#### `Login.jsx`

Компонент формы входа.

**Состояния:**

- `formData` - данные формы (email, password)
- `loading` - флаг загрузки
- `error` - сообщение об ошибке

**Функциональность:**

- Валидация полей формы
- Отправка запроса на вход
- Сохранение токена в localStorage

#### `Register.jsx`

Компонент формы регистрации.

**Состояния:**

- `formData` - данные формы (email, password, confirmPassword)
- `loading` - флаг загрузки
- `error` - сообщение об ошибке

**Функциональность:**

- Валидация полей формы
- Отправка запроса на регистрацию
- Сохранение токена в localStorage

### Компоненты сессий

#### `SessionList.jsx`

Компонент для отображения списка сессий.

**Функциональность:**

- Загрузка списка сессий с сервера
- Отображение сессий в виде карточек
- Фильтрация сессий по статусу
- Обработка событий WebSocket для обновления списка в реальном времени

#### `SessionCreate.jsx`

Компонент для создания новой сессии.

**Состояния:**

- `formData` - данные формы (videoLink, startTime)
- `loading` - флаг загрузки
- `error` - сообщение об ошибке

**Функциональность:**

- Валидация полей формы
- Отправка запроса на создание сессии

#### `SessionJoin.jsx`

Компонент для присоединения к сессии и выбора роли.

**Состояния:**

- `selectedRole` - выбранная роль
- `loading` - флаг загрузки
- `error` - сообщение об ошибке

**Функциональность:**

- Выбор роли (interviewer, interviewee, observer)
- Отправка запроса на назначение роли
- Обработка ошибок (роль уже занята, нет прав и т.д.)

#### `RoleSelector.jsx`

Компонент для выбора роли в сессии.

**Функциональность:**

- Отображение доступных ролей
- Обработка выбора роли

### Компоненты обратной связи

#### `FeedbackForm.jsx`

Компонент формы обратной связи.

**Состояния:**

- `formData` - данные формы (ratings, comments, recommendations)
- `loading` - флаг загрузки
- `error` - сообщение об ошибке
- `success` - флаг успешной отправки
- `activeSection` - активный раздел формы ('ratings', 'comments', 'recommendations')

**Функциональность:**

- Многоэтапная форма с оценками, комментариями и рекомендациями
- Валидация полей формы
- Отправка запроса на сохранение обратной связи

#### `FeedbackResults.jsx`

Компонент для отображения результатов обратной связи.

**Функциональность:**

- Загрузка обратной связи для сессии
- Отображение оценок, комментариев и рекомендаций
- Отображение статуса заполнения обратной связи обеими сторонами

### Контексты и хуки

#### `SocketContext.jsx`

Контекст для работы с WebSocket соединением.

**Состояния:**

- `socket` - экземпляр Socket.IO
- `connected` - флаг подключения
- `error` - сообщение об ошибке
- `reconnectAttempts` - количество попыток переподключения

**Функциональность:**

- Установка WebSocket соединения
- Обработка событий подключения, отключения, ошибок
- Предоставление методов для работы с сессиями (joinSession, leaveSession)

#### `useSocket.js`

Хук для использования SocketContext.

**Функциональность:**

- Предоставляет доступ к состоянию и методам SocketContext
- Упрощает использование WebSocket в компонентах

## Взаимодействие компонентов

1. **Аутентификация:**

   - `App.jsx` отображает `Login.jsx` или `Register.jsx` для неаутентифицированных пользователей
   - После успешной аутентификации `App.jsx` отображает `UserProfile.jsx` и `SessionManager.jsx`

2. **Управление сессиями:**

   - `SessionManager.jsx` управляет отображением компонентов сессий
   - `SessionList.jsx` отображает список сессий и позволяет выбрать сессию для присоединения
   - `SessionCreate.jsx` позволяет создать новую сессию
   - `SessionJoin.jsx` позволяет присоединиться к выбранной сессии и выбрать роль

3. **Обратная связь:**

   - После завершения сессии пользователи могут оставить обратную связь через `FeedbackForm.jsx`
   - Результаты обратной связи можно просмотреть через `FeedbackResults.jsx`

4. **WebSocket:**
   - `SocketProvider` оборачивает компоненты, которым требуется WebSocket соединение
   - Компоненты используют `useSocket` хук для доступа к WebSocket функциональности
   - WebSocket используется для обновления списка сессий, уведомлений о выборе роли и обратной связи в реальном времени

## Технологии

- **React** - библиотека для создания пользовательских интерфейсов
- **Vite** - инструмент сборки и разработки
- **Socket.IO** - библиотека для работы с WebSocket
- **TailwindCSS** - утилитарный CSS-фреймворк для стилизации
